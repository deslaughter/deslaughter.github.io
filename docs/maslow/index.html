Maslow CNC #

The Maslow CNC is a system for cutting large pieces of thin material, such as a 4x8 sheet of plywood, with reasonable accuracy for minimimal cost.

The plans and source code for the Maslow CNC are available on the MaslowCNC github site.

Components #

The Maslow CNC has four major compontents: frame, sled, actuators, and controller.
